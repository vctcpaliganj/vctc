Adobe Photoshop 7.0

It is a picture editing software that is generally used to modify and design the picture of different extension such as jpg, png, bimp, psd, etc.
Generally there is no size limitation of the picture in Photoshop.

• The default extension of Photoshop picture is .psd but we can save the picture in different extension using “save as type”/ “format” option.

• Basic features of Photoshop:

1. It provides cleaning tool to remove the dark spot from the picture.
2. We can easily merge and split the picture.
3. Color correction feature is also provided to change the color of picture.
4. Layer pallete is also provided to work seperatelly on the picture.
5. Text related work can also be done in Photoshop.
6. We can also create poster in Photoshop.

COMPONENT OF PHOTOSHOP WINDOWS

1. Title bar:-

Generally it display the details of picture.

a. Programme name
b. File name with extension.
c. Zoom percentage
d. Color mode ( RGB, CMYK, grayscale)

RGB = Red Green Blue
CMYK = Cyan Magenta Yellow Black
Grayscale = Black/White

2. Menu bar:-

There are 9 menu in the Menu bar.
1. File
2. Edit
3. Image
4. Layer
5. Select
6. Filter
7. View
8. Windows
9. Help

3. Properties bar:-

It lies just below the menu bar. It display different options(tool) related to the selected tool from tool box .

4. Tool box:-

To the left side of screen 22 tools are given to perform any task in Photoshop.
A tool having arrow in it , has more options of tool.

5. Layer pallete:-

To the right side of the screen layer pallete is given that helps to perform layer related work.

a. Whenever we open multiple picture is blank document then multiple layer is created in layer pallete .
b. To perform any task of the picture, we need to select the layer.
c. To change the size position and rotation of picture press [ ctrl + t ].
d. If we open any one picture using [ctrl + o] then this picture will be open as background layer.

To change the background into layer double click on it.

6. Color Box:-

There are two types of color.
a. Foreground color
b. Background color

1. Color box lies just below the tool box.
2. We can create custom foreground can we inverse by double clicking in the color box.
3. Foreground and background can we inverse by pressing key (x).
4. Foreground color is drawing color and Background color is eraser color.

7. Status Bar:-

The bottom area of the Photoshop windows is called status bar. It display the zoom percentage and the size of the picture.

Note:
All the above component of Photoshop Windows can be show or hide using Windows menu.

TOOLBOX OF PHOTOSHOP 7.0

A. Selection / Marquee:- [ m ]

There are two types of selection that we regularly use:-
1. Rectangular selection
2. Elliptical selection

Note:
1. We can either select single or multiple part of picture by using.
2. We can also select multiple part of picture by holding shift key.
3. We can create a copy of selected part of picture by dragging the mouse with alt key.
4. We can also set feather point from property bar/option.
5. To Deselect the selected area press [ ctrl + d ]
6. If we use multi selection mode in over lapping condition. Then a new path is created and selection is done according to created path.

B. Move tool:- [ v ]

Using this tool we can move the selected part of picture by just dragging the mouse.

1. After moving the picture background color is created filled in the previous selected area.
2. If we use move tool in to other windows than copy is created.

C. Lasso tool:- [ l ]

There are 3 tool in its. That helps to make the costume selection by dragging the mouse.
a. Lasso tool
b. Polygonal Lasso tool
c. Magnetic Lasso tool

D. Magic wand tool:- [ w ]

This tool is used to select the solid color from the picture by just clicking on it.

a. We can also use single or multimode to select the picture.
b. Before using magic wand tool we should set tolerance value.
c. To fill color in the selected area, press shift + backspace and select the color mode and opacity.
d. To inverse the selection press [ ctrl + shift + I ]

E. Crop tool: [ c ]

It is used to crop the picture after cropping the unwanted area of picture is removed.
Before using cropping tool we should clear the field of width and height from property bar.

Note: If we want to crop the picture at definite size. Write the value of width and height in property bar

Steps to create passport size of photo:-

1. Open the picture and then select crop tool.
2. Set width ( 1.2 inch ) and ( 1.8 inch ) in property bar and select the picture.
3. Press Enter and set inside stroke from edit menu.
4. Press [ ctrl + b ], [ ctrl + m ], [ ctrl + l ] to adjust the color and copy the picture.
5. Take a new blank document and paste the copy picture.
6. Make dublicate of the picture to fill the page and finally link all the layer by using [ ctrl + e ].

F. Knife tool: [ k ]

Using this tool we can split the different part of picture by following steps.

1. Select knife tool and mark the area of picture.
2. Press [ ctrl + alt + shift + s ] then click on save.
3. Right file name, select location and press enter.
4. A new folder is auto created in which the part of picture is sorted.

G. Healing Brush tool / path tool:- [ j ]

This tool is used to remove the dark spot from the picture by two alternative method.

Process: 1
1. Select healing brush tool and then click on the clean area of the picture, with [ alt ] key.
2. Click on the dark spot we want to remove.

Process: 2
1. Select patch tool and then select the dark spot from the picture.
2. Drag the select part of the clean area of picture.

H. Brush tool:- [ b ]

Using this tool we can fill color in the selected part of picture by dragging the mouse.

Note:
1. Brush use foreground color for fill.
2. Before using brush, we should select brush, we should select brush size, mode, and opacity from property bar.

I. Clone tool :- ( s)

a. Clone stamp tool:-

Using this tool we can create a dublicate copy of part of picture by following.

Steps:
1. Select clone stamp tool and then click on the part of picture with [ alt ] key.
2. Move the mouse on the another part of the picture where we want to create the dublicate copy.

b. Pattern stamp tool:

To fill texture effect in the select part of picture.
To use this tool we must select texture from property bar and change the brush size and opacity of the select texture effect.

We can create a new pattern by clicking on >> and select “load pattern” option.

J. History brush tool:- [ y ]

It remove all the work of picture and change the picture in its original form.

K. Eraser tool:

There are 3 types of eraser that help to delete the part of picture.

1. Eraser: To delete the part off picture by filling the background color.
2. Background Eraser tool: To delete the part of picture by completely removing the background.
3. Magic Eraser tool: It delete the part of picture according to the solid color criteria.

L. Paint Bucket tool/Gradient tool: [ g ]

Using this tool we can fill solid color in the selected area of the picture.
We can also fill gradient (mixture of color) in the area of picture.

Note:
1. Using paint bucket tool foreground color is filled.
2. In case of gradient we need to select gradient style, opacity and its mode.
3. We can also load more gradient by clicking >> ( symbol ) from property bar.

M. Blur tool/sharpen/smudge tool: [ r ]

To apply blue effect and smudge effect in the picture.
By using smudge tool, any part of picture smudge tool, any part of picture can be move by dragging the mouse.

N. Dodge tool/Burn tool/Sponge tool: [ o ]

To apply whitening effect, Raddish effect, orange effect, color effect in the picture respectively.

O. Text tool: [ t ]

There are 4 tool in its.

a. Horizontal type tool:
To write any text from left to right direction.

b. Vertical type tool:
To write any text from top to bottom direction.

c. Horizontal type mask tool.
Using this tool we write text from left to right show that the selection of the picture is taken according to the written text.

d. Vertical type mask tool.
Using this tool we write text from top to bottom shot that the selection of the picture is taken according to written text.

Note:
1. To perform the text related work, press [ ctrl + t ] after selecting the text.
2. Using property bar we can change the size, style, color and alignment of the text.
3. The text can be convert into different shape using text Arc option from property bar.
4. We can apply different effect in the text such as shadow, reflection, glow stroke, gradient etc. using Blending option after right clicking on layer.
5. In case of type mask tool, we write the text to select the part of picture according to the text.

Steps to use type mask tool:

1. Open any picture and write any text on it. Using type mask tool.
2. Modify the size style, arc etc. and click on accept icon.
3. Press [ ctrl + c ] and create a new document by pressing [ ctrl + n ].
4. Press [ ctrl + v ] and resize it using [ ctrl + t ].

P. Pen tool: [ p ].

This tool also helps to make the selection of any part of the picture by creating a path.
After creating the path we need too right click and then select “make selection” option.

Note:
1. Using “add anchor point” option of this tool. We can add notes in path to customize .
2. The nodes can also be delete using delete anchor point option.

Q. Shape tool: [ U ]

There are different types of options that helps to create different types of shape such as:-
Rectangle, Rounded Rectangle, Ellipse ,Polygon, Line of any custome shape.

Note:
1. Before creating any shape we must select shape tool from property bar.
2. In case of pen tool or shape tool we need to select any one option. From property bar.
a. Shape option. ////////image
b. Path option . /////////// image

R. Notes/Audio annotation tool: [ n ].

Using this tool we can write some additional text for the picture as a comment.
We can also insert audio clip by recording our voice.

Note:
The nodes of audio annotation can be view or delete by right clicking on it.

S. Color picker/eye dropper: [ I ]

To pick color form any part of picture.

Note:
The picked color is always displayed in foreground color.

T. Hand tool: [ H ]

This tool can be used only if picture is view in larger form.
Using this tool picture can be move anywhere on screen.

U. Zoom:

To view the picture in larger of smaller form.
