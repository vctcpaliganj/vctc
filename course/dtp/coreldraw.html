Corel Draw

It is highly advanced graphics related software that helps to create different types of Document such as: Posters, Banners, Identity Card colorful booklet, different types of LOGO.
Generally multi color publishing work is done in Corel Draw.
LOGO is pictorial representation of any compony name or institute name.

System Configuration to install Corel Draw.

a. The minimum storage of RAM must be 4GB.
b. Installed Windows must be Window 7,8,10 & 11.
c. Operating System must be of 64Bit.
d. Processor of the PC must be i3 , i5, i7
e. The available storage of hard disk must be at least 3 GB to install the application.

The 1st screen that is open after starting CorelDraw is called Welcome Screen.

There are different options provided in welcome screen.
1. New Document
2. Recently created document of Corel Draw
3. Open Document option.

1. Title bar

Tool box of Corel Draw 2021
1. Pick tool

This tool is used to select the object of document.
a. If we select this tool and does not select any object. Then property of page is displayed in property bar.
b. If we select the object then property of selected object is displayed in property bar.
c. For multiple selection of object, use shift key.
d. To Rotate the created shape double click on the created shape.
e. Drag the shape and leave it with right click to create multiple copy.
f. To repeat the created shape press [ ctrl + r ].

2. Shape tool

This tool can be use only after creating any shape in the document.

There are different options in it. That helps to customize the created shape.
a. Smooth: To make the corner ( rounded ).
b. Smear : To drag the line inside or outside.
c. Twirl : To apply spiral effect.

Note:
To customize the shape manually, select the shape and convert it into curve [ ctrl + q ]

3. Crop tool

It is used to crop the picture
• To insert picture in Corel Draw press [ ctrl + I ]
• Knife tool is also given in crop that helps to cut the object in two parts.
• Virtual segment delete : it is used to delete the part of shape.

4. Free hand tool [ F5 ]

There are different options in it that helps to create line in the document.
a. Free hand : To create custom drawing as a pencil.
b. To point line: To create a line.
c. Pen tool: To create custom shape by meeting the point of line.

5. Zoom tool [ z ]

Using this tool, the created document can be view in larger or smaller form.
There are different options provided in the property bar to zoom.
a. Custom zoom
b. Zoom in [ ctrl + + ]
c. Zoom in [ ctrl + - ]
d. Zoom to select [ shift + F2 ]
e. Zoom to all object [ F4 ]
f. Zoom to all page
g. Zoom to page [ Shift + F4 ]
h. Zoom to page width
i. Zoom to page height

6. Rectangle [ F6 ]

To create Rectangular shape in the document.
a. Using property bar the size of the created rectangle can be change.
b. The corner of rectangle can be change using 3 options of property bar.
///// image ///
c. The line style and size can also be changed using property bar.

7. Ellipse Tool [ F7 ]

1. Before creating Ellipse we must select Ellipse, pi, arc from property bar and write the value degree for arc or pie.
2. Any object can be repeat on created path (shape) by selecting “fit object to path” option from object menu.
3. Incase of text it can also be fit on created path by using “fit text to path”. Option from text menu.

8. Polygon shape

There are different types of shape in this tool such as.
a. Polygon
b. Star
c. Spiral
d. Common shape

Note
1. In case of polygon we need to write number of sides in property bar.
2. In case of star, we need to select star /// image or complex star /////// image from property bar.
3. We can also add start arrow head and end arrow head from property bar.

9. Artistic Media tool

Using this tool we can create different types of colorful style as the border of page.

Note
Using property bar we can select different types of brush style .

10.

a. Text tool

Using this tool we can write any text in the page.
There are two types of text that can be written in document
a. Artistic tool
b. Text in a text box ( Normal text )

Note
1. After selecting text tool, click in the document to artistic text and drag in the document to write normal text
2. In case of Artistic text, we select the text using pick tool and in case of normal text we select the text using text tool.
3. Text related work can be done using property bar text menu of [ ctrl + t [
4. The Artistic text can be convert into curve [ ctrl + q ] and can be customize manually using shape tool.
5. Alignment ,Indent , Bullet ,Numbering can’t be apply in Artistic text.

b. Table

This tool is used to create table in the document .
1. Before crating table we need to write number of row and column in property bar.
2. The outline color and fill color can also be change from propertybar.
3. All the table related work can be done using property-bar or table-menu such-as;
a. Select cell / Row / column / table
b. Insert Row / column
c. Delete Row / column
d. Merge [ ctrl + M ]
e. Split Row / column
f. Distribute row/column Evenly

11. Connector tool

To connect two shape together shape can be connected using straight line angle connector and rounded angle connector.

12.

a. Contour tool
Using this tool, we can create multiple copy of selected shape inside or outside the created shape.
We need to write the number of copy and select inside or outside option from property bar.

b. Blend
This tool is used to create multiple object by dragging the mouse from one shape to another
The number between two shape can be change from property bar.

c. Distort
Using this tool the created shape can be change into different shape such as pull corner, push corner twister stamp zipper.

13. Transparency tool

Generally this tool is used to apply transparency effect in the picture so that it is visible on text.
Generally we used this tool to move the content picture or shape behind the text.

14. Color picker

To pick color from the picture.

15. Intersective fill ( Gradient ) [ g ]

It is used to fill solid color, gradient or pattern in the created shape. After selecting this tool different types of pattern and gradient are displayed to the property.
