It is an application of DTP package that is used to create the different types of document without document document size limitation.

The default extenstion of Pagemaker is (.pmd)

Whenever we open Pagemaker application There is no any document.

To create the document we press [ ctrl + n ] and then setup the coucment.

Version of Pagemaker:-

There are mainly 7 Version of Pagemaker Pagemaker 7.0 is the last version of Pagemaker.
After this version Adobe illustrator application is launched.

Steps to install Adobe Pagemaker.

1. Copy the Pagemaker package from any other (PC) and paste it in your (PC).

2. Open the folder of Pagemaker and double click on "setup.exe".

3. Continue click on next until get activation windows.

4. Fill the basic details such as salutation, firstname and serial No of 24 digit.

5. Clikc on next and finally click on Finish.

Steps to create multiple id-card in pmd

1. Open excel and create the required data in it.

2. Save the data of excel in .csv(macintosh) format and save it.

3. Now open pagamaker and create a new document by taking proper size of id card. 2.2 x 2.8

4. Create a format of id card according to your needs.

5. Show data merged pallete from plug-in pallete of windows menu.

6. Click on the arrow of merge pallete and select data source option.

7. Select the created data of excel and press enter.

8. Place the cursor in created format of page maker and insert the required field.

9. Finally click on arrow and select data merged option.
