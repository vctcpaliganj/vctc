Microsoft Paint

It is a drawing related software that helps to modify the picture and create any drawing.

• The picture of Paint can be saved in different extension such as .jpg, .bmp, .png, .gif using “save as” type option.

Process to open picture in Paint.

1. Right click on the picture and then select edit option.
2. Open Paint press [ ctrl + o ] select picture and press enter.
3. Open Paint click on “paste from” select picture and press enter.
4. Right click on the picture and then select open with option then select Paint to open.

There are 3 menu in the menu bar to perform the picture related work.

1. File menu
2. Home menu
3. View menu

File menu

There are different option in this menu in which some options are same as Wordpad.

1. New [ ctrl + N ]
2. Open [ ctrl + O ]
3. Save [ ctrl + S ]
4. Save as [ F12 ]
5. Print [ ctrl + P ]
6. From Scanner
7. About Paint
8. Set as Desktop Background
9. Properties [ ctrl + E ]
10. Exit [ alt + F4 ]

• From Scanner:-

This option can be used to scan the image and directly bring the picture in Paint.

• Set as Desktop Background:-

Using this option any picture can set as the desktop background in 3 form.

1. Fill
2. Center
3. Tiles

Note:
1. Before Setting any picture as a desktop background we need to save it.
2. To set any picture as tiles, the size of picture must be small.

• Properties: [ ctrl + E ]

It display the complete details of the picture such as size, height, width etc.

Home menu

This menu is divided into 5 submenu that help to perform the picture and drawing related work.

1. Clipboard menu:-

There are 3 options in it
1. Cut [ ctrl + X ]
2. Copy [ ctrl + C ]
3. Paste [ ctrl + v ]

Note:
1. To copy or cut any picture need to select it using selection tool.
2. Once copied or cut picture is always past to the top left corner.
