It is a drawing related software that helps to modify the picture and create any drawing.

• The picture of Paint can be saved in different extension such as .jpg, .bmp, .png, .gif using “save as” type option.

Process to open picture in Paint.

1. Right click on the picture and then select edit option.
2. Open Paint press [ ctrl + o ] select picture and press enter.
3. Open Paint click on “paste from” select picture and press enter.
4. Right click on the picture and then select open with option then select Paint to open.

There are 3 menu in the menu bar to perform the picture related work.

1. File menu
2. Home menu
3. View menu

File menu

There are different option in this menu in which some options are same as Wordpad.

1. New [ ctrl + N ]
2. Open [ ctrl + O ]
3. Save [ ctrl + S ]
4. Save as [ F12 ]
5. Print [ ctrl + P ]
6. From Scanner
7. About Paint
8. Set as Desktop Background
9. Properties [ ctrl + E ]
10. Exit [ alt + F4 ]

• From Scanner:-

This option can be used to scan the image and directly bring the picture in Paint.

• Set as Desktop Background:-

Using this option any picture can set as the desktop background in 3 form.

1. Fill
2. Center
3. Tiles

Note:
1. Before Setting any picture as a desktop background we need to save it.
2. To set any picture as tiles, the size of picture must be small.

• Properties: [ ctrl + E ]

It display the complete details of the picture such as size, height, width etc.

Home menu

This menu is divided into 5 submenu that help to perform the picture and drawing related work.

1. Clipboard menu:-

There are 3 options in it
1. Cut [ ctrl + X ]
2. Copy [ ctrl + C ]
3. Paste [ ctrl + v ]

Note:
1. To copy or cut any picture need to select it using selection tool.
2. Once copied or cut picture is always past to the top left corner.

2. Image Subment:-

There are four tools in it.

A. Select :

There are 3 types of selection to select picture.
1. Rectangular selection.
2. Free form selection.
3. Select all.

B. Resize: [ ctrl + w ]:

It is used to change the size of picture in two form.
1. Percentage wise
2. Pixel wise ( 500 x 400 )

• Note: using ( skew ) option we can also Orient the picture at any angle between 0 to 89.

c. Crop : [ ctrl + shift + x ]:

It is used to crop the selected part of picture after cropping the selected part of picture remain save and rest parts get deleted.

D. Rotate:

To rotate the picture in different direction by selecting following options.
1. Rotate 90 degree right.
2. Rotate 90 degree left.
3. Flip vertical
4. Flip Horizontal.
5. Rotate 180 degree.

3. Tools Submenu:

There are 6 six tools in it.

1. Pencil :

a. This tool is used to create custom drawing by dragging the Mouse.

Note: Before using pencil tools we should select pencil size and color. (Color 1)

b. To change the size of pencil of Erase we press ( ctrl + +/- ) of numeric key.

2. Fill with color ( Bucket ) :

it is used to fill color in the closely bonded picture.

Note: If picture is not bonded properly the filled color become background color.

3. Text tool:

It is used to write any text on the picture.

Note:
1. The size , style, effect and color of the text can be change.
2. The text can be edit until cursor blink there.

4. Eraser:

It is used to delete the part of picture by dragging the mouse.
Color2. It is the color of Eraser that must be save as background.

5. Color Picker:

It is used to pick color from the picture and use in drawing.

6. Zoom:

To view the picture in large or Smaller form.
Zoom in -> Click.
Zoom out-> Right Click.

4. Shape Submenu:

There are different type of shape in the shape box that helps to create ay drawing easily.

Note:
1. Before create any shape we must select outlining size color and then drag the mouse.

2. Straight line and square is not given in the shape box to draw it we use.

a. Shift + oval = circle
b. Shift + Rectangle = Square
c. Shift + line = Straight line.

Curve line looks like straight line, it can be change in to curve by dragging the mouse from line.

5. Color Submenu

There are different color in the color box that is used to create color full drawing.

color 1 is the drawing color and color 2 is the background color.
We can also create a new color by click on edit color option.
Using ( full screen ) option view menu is used to view create picture as full screen.

View Menu

There are 3 submenu in it

1. Zoom:

a. Zoom in:
Its zoom the image in larger form.

b. Zoom out:
Its zoom the image in smaller form.

c. 100% :
Its show the image into 100%.

2. Show or hide

a. Rulers: [ ctrl + r]
To show or hide rulers.

b. Grideline:[ ctrl + g ]
To show or hide grideline.

c. Status bar:
To show or hide status bar.

3. Display:

Full Screen:[ F12 ]
To show the image in full screen.
