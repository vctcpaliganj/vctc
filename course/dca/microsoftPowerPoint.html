It is presentation software in which we create data with motion and sound.

To create any Presentation in Power Point be need to follow 4 steps.
1. Create the content of Presentation as similar to Word.
2. Design the slide and its content.
3. Insert Audio or video click in the slide.
4. Add Transition and Animation effect and finally press F5.

Basic concept of Power Point.

1. The page of Power Point is called Slide. And its document is called Presentation.
2. The colorful design of the slide that appear during opening Power Point is called Theme.
3. The small form of slide that appear to the left side of screen is called Thumbnail.
4. The rectangular box of the slide is called Place Holder in which we can write any text.
5. In case of blank slide, we need to Insert WordArt. Textbox of Shape to write any text.

Use of Power Point:-

a. To attend line meeting.
b. To show report.
c. To create advertisement content.
d. To create short Video. Etc.

Home Menu

1. Clipboard Sub Menu

There are 4 tools
a. Cut ( ctrl + x )
b. Copy ( ctrl+ c )
c. Paste ( ctrl + v )
d. Format Painter ( ctrl + shift + c/v)

Note:-
The working process of all four options are same as Word.

2. Slide Sub Menu

There are three important options
a. New slide ( ctrl + m )
To create a new slide according to our needs.

b. Layout
To change the type of slide without changing its content.

c. Reset
It remove all the design of Place Holder and change the Place Holder in its original form.

3. Font Sub Menu

There are different tools to perform the text related work.
a. Font style
b. Font size
c. Increase font size
d. Decrease font size
e. Clear formatting.
f. Bold
g. Italic
h. Underline
i. Strike through
j. Change case
k. Font color
l. Character spacing
m. Shadow effect

Character spacing

There are 5 types of character spacing that helps to adjust the space between two letter.
a. Normal
b. Loose
c. Vert Loose
d. Tight
e. Very Tight

Note:-
To set the custom space between letter, select more spacing & than select “Condense Expand” write the value.

Shadow Effect

To apply shadow effect in the selected text.

Note:-
a. There is only single underline.
b. To apply Super Script and Sub Script, press [ ctrl + shift + f ] & select the required option.

4. Paragraph Sub Menu

There are different tools to arrange the text.
a. Alignment
Left, Right, Centre, Justify

b. Indent
There are two Indent
1. Left Indent
2. Right Indent

c. Line spacing
To give the space between line.

d. Bullet & Numbering
To apply Bullet & Numbering in the of place holder.

e. Column
It is used to split the place holder in two or more column.

Note:-
The place holder can be split in maximum 16 column in Word 45 column.

f. Text Direction
It is used to change the direction of Text.

g. Align Text
To align the text to the Top, Middle, or Bottom of the place holder.

h. Convert the smart Art
To convert the text in the form of Smart Art.

5. Drawing Sub-Menu

This Sub-Menu is used to perform the shape related work.
There are different options provided in it.
a. Shape Box
b. Shape Style
c. Shape Fill
d. Shape Effect
e. Shape Outline
f. Arrange

Note:-
Shape related work can also be done using Insert Menu and Format Menu.

6. Editing Sub-Menu

There are 3 options in it
a. Find [ ctrl + F ]
b. Replace [ ctrl + H ]
c. Select All

Note:-
We can also replace the Font use in the slide by Replace option.

• Photo Album

This option is used to create photo album in PowerPoint by following steps:-
1. Click on Photo Album option from Insert Menu.
2. Select New Photo Album Option.
3. Click on File / Disk & then select multiple picture with [ ctrl key ].
4. Modify and arrange the picture, select Number of picture for slide using “Layout” & finally click on create.

Note:-
1. After creating Album click on Design Menu than Design the slide.
2. The picture and text of Album can be designed using Format Menu.
3. We can also add Music in the Album using Audio option.
4. After giving final touch to Album apply transition and animation effect.
5. We press F5 key to show the slide.

• Audio

This option is used to insert audio clip(भाग) in the slide.

Note:-
1. After inserting Audio click on “Play in background” option from playback-Menu.
2. The inserted Audio can be Trim using playback Menu.

• Video

This option is used to insert video in the slide

Note:-
1. Using “Video shape” option the inserted video can be change into any shape.

2. Using poster Frame option, we can place any picture on the video.

3. If we want to play the Video automatic during slide show click on automatic option from playback menu.

4. As similar to Audio, inserted video can also be trimed.

Design Menu

This menu is used to change the setup of slide and also helps to design the slide with the help of following steps:-

1. Theme

There are different types of pre-designed style for the slide we can select any one theme.

2. Variants

The 4 different color of the theme is called variant.

3. Slide Size

It is used to change the size and orientation of slide.

Note:-
We always select “Ensure” option to set the slide size and orientation.

4. Format Background

This option is used to fill color, gradient, texture, pattern, picture in the slide.

Transition Menu

This menu is generally used to apply motion effect in the slide.
There are different options in this menu that helps to apply transition effect in the slide.

1. Transition effect

There are different types of transition effect such as:- Ripple, Curtain, Fracture, Push, Wipe, Reveal, Crush, Glitter, etc.
We can select only one transition effect.

2. Effect

It is used to change the direction of applied transition effect.

3. Duration

It is used to set the time to display the slide during slide show.
If time is more than speed will be slow.
If we want to show the slide automatic than select “After” option in the place of Mouse click.

Note:-
1. We can apply same of different transition effect in all the slide.
2. We can also add Popup Music to display the transition effect.

Animation Menu

This menu is used to apply motion effect in the content of slide.
We need to select the content to apply animation effect.

There are 4 Types of Animation.

1. Entrance

This Animation is used at first to appear the content on screen.

2. Emphasis

This Animation is used after Entrance to add more effect in the content.

3. Exit

This Animation is used at last to remove the content from screen.

4. Motion Path

It is additional from screen Animation effect that helps to move the content on a fixed path.

• All the applied animation of the slide can be view or manage by clicking on Animation Panel option.
• To add more animation in the content, click on “Add animation” option.
• As similar to transition, the direction and shape of animation can also be managed.

Slide Show

Generally we press [ F5 ] key to show the Animated slide.
Using different options of this menu the slide can be show.

1. Start slide show

We can show the slide in 3 form.
a. From beginning [ F5 ]
b. From current [ shift + F5 ]
c. Custom

2. Hide slide

To show or hide the slide during slide show

3. Setup slide show

Using this option we can set the slide number to show.
We can also show the slide without animation.
