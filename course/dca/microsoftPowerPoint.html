Microsoft PowerPoint

It is a presentation software that is used to create different types of document with motion and sound. The extension of POwerPoint document is .ppt.

There are many used of Power Powerpoint such as:-

1. To create advertisement content.
2. To create project work.
3. To make project work.
4. To attend live meeting ( webinar )
5. To create short video.

Basic concept of PowerPoint: -

a. The document of PowerPoint is called presentation and its page is called slide.

b. By default, there is one slide in presentation that can be extend by pressing [ ctrl + M].

c. There are different types of predesigned slide-colorful is called them.

d. All the slide is displayed to the left side of screen is called thumbnail.

e. Each slide has a rectangular box, is called place holder.

f. If we choose blank slide without any place holder than we cannot write any text in it.

To write any text, we should insert text box or word Art.

g. All the work in PowerPoint is done same as word, but we can add motion and sound in it.

h. Generally we create any document in PowerPoint for presentation purpose.

About Menubar

1. File menu: - New, open, save, save as….

2. Home menu: - To perform text, shape and slide related work.
3. Insert menu: - To add any object

4. Design menu: To perform the design related work of slide and change the size of slide.

5. Transition menu: - To add motion in the slide.

6. Animation menu: - To add motion in the content of slide.

7. Slide show menu:- To show the animated slide.

8. Review menu:- Spelling check , Antonyms synonyms ……….. etc.

9. View menu: - To view the slide in different form.
