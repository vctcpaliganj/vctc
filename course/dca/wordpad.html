Wordpad

It is another text editing software that is much better than notepad, it is documentation software in which we can create different types of document such as letter , Notice, Assignment, application question paper etc.

ADVANTAGE OF WORDPAD

1. We Change the size , style , color of any selected text.
2. The default color of text is black that can be change is any color.
3. We can also arrange the text to the left, right of centre of the time.
4. The work of page setup [ margin ,size orientation ] is visible in page.
5. We can insert any picture in the page of document.

Note:
The default extension of wordpad file is .rtf that can also be change be clicking on .save as type” option

Ccomponent Wordpad Window
1. Title bar:-

it lies to the top of screen to display the programeme name and file name.

2. Menu bar:-

There are three menu in the menu bar.

a. File menu
b. Home menu
c. View menu

3. Ribbon [ctrl +F1]:-

it lies just below the menu bar to show or hide all the tools of selected menu.

4. Status bar:-

To the right side of status bar zoom panel is found that helps to view the document in larger of smaller form.

File Menu

There are different options in this menu in which most of the options are same as notepad.

1. New
2. Open
3. Save
4. Save as
5. Page setup
6. Print
7. Exit
8. About wordpad : It display the install version .
9. Send as mail: To share the of wordpad document using Internet.

Home Menu

It is the important menu of Wordpad. From where all the word done. This men is divided into five sub-menu.

1. Clipboard sub-menu:-

There are 3 tool in it.
a. Cut
b. Copy
c. Paste

2. Font sub-menu:-

There are 12 tools in this sub-menu that helps to perform the text related work. ///////////////// image//////////////////

1. Font style:-
To change the writing style of text.

2. Font size:-
to change the size of selected text.
The maximum font size is 1638 and minimum is 1%

3. Grow font size:-
To increase the size of text. Ctrl + shift + >

4. Shrink font size:-
To decrease the size of text. Ctrl + shift + <

5. Bold:-
To make the selected text bold.

6. Italic:-
To make the selected text Italic.

7. Underline:-
To underline the selected text.

8. Strike through:-
Using this tool we can draw a horizontal line across the selected text.

9. Sub-script [ ctrl + “ + “ ]:
It helps to move the selected text to the top of normal text.
Example: H2so4 , kno3 etc.
Example: ( x + y )2 , x2 + y2 =0 , 10th etc.

10. Super script: [ script + shift + “ + “ ]:
it helps to move the selected text to the top of Normal text.
Example: ( x + y )2 , x2 + y2 =0 , 10th etc.

11. Font color:
it is used to fill color in the selected text.

12. Highlighted color:-
It is used to fill color behind the text.

3. Paragraph Submenu

There are 9 tool in this sub-menu that helps to arrange the text. /////////////// image ////////////

1. Alignment : -

There are 4 types of alignment that directly move the text to the left , Right or center of the line.
1. Left alignment [ ctrl + L ]
2. Right alignment [ ctrl + R ]
3. Centre alignment [ ctrl + E ]
4. Justify alignment [ ctrl + J ]

2. Indent :

There are two types of indent that helps to move the text to the left or right step-by-step.
1. Left indent
2. Right indent

c. Line spacing :-

This tool is used to adjust space between two lines.

Note:-
To remove the line spacing completely select [ 1. 0 ] and then click on last option.

d. Bullet or Numbering :-

This option is used to automatic apply the serial number in each line of document. When press enter next numbering auto appear.

Note:
The size and color applied Bullet of numbering is same as font size and color.

e. Paragraph tool:-

It is store house of all the tools of paragraph . We can perform all the arrangement of text with this tool.

4. Insert Submenu

There are 4 tool in this sun-menu.

a. Picture:-

The option is used to insert any picture in the document of word-pad.

Note: The size of picture can be change by two method.
1. Drag the mouse from corner of picture.
2. By selecting “ resize picture” option from picture tool.

Note: As similar to text picture can be align to the left, right or centre of line.

b. Paint drawing: [ ctrl + d ] :

By using this option , we can insert picture picture in wordpad by creating the picture in paint. When we click on this option Ms. paint is automatic open where we create our drawing.

Note: To modify the picture of paint click on inserted picture.

c. Date and time:-

This option is used to insert current date and time in the document where cursor blink.

d. Insert object:-

Using this option we can insert the document of any other programme.
such as :- word, excel, powerpoint, in wordpad,

There are two option:
a. Create new
b. Create from file.

5. Editing Submenu

There are three options in it.
a. Find
b. Replace
c. Select all

View Menu

This menu is generally used to view the document in different form.

This menu is divided into three sum-menu.

1. Zoom:-

To view the document in large of smaller form.
a. Zoom in ( + )
b. Zoom out ( - )
c. Normal ( 100% )

Note:- The document view on 100% is printed as same as screen size.

2. Show or hide:-

To show or hide rular ( scale ) and status bar from the screen.

a. Word wrap:- There are three options.

1. No wrap: - All text in single line.
2. Wrap to ruler:- All text is visible as page.
3. Wrap to windows:- All text is visible as full screen.

b. Measurement unit:- To change the unit of scale
