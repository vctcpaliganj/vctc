Notepad

It is a text editing software in which can simply write any text without designing the text.
Generally the command of HTML ( hyper text markup language ) is written in notepad.

The default extension of notepad file is ( .txt ) .

Note:
Extension is a code that helps to identify the types of file.

1. Notepad = .txt
2. Wordpad = .rtf
3. Paint = .bmp/ .png / .gif
4. Picture = .jpeg/ .jpg
5. Music = .mpeg3
6. Video = .mpeg4 / .avi
7. Ms. word = .dox
8. Ms. excel = .xlsx
9. Ms. powerpoint = .pptx
10. Page maker = .pmd
11. Photoshop = .psd
12. Corel draw = .cdr
13. Application of mobile = .apk
14. Application of window = .exe

Limitation on notepad.

a. We can change the size and style of text cannot change the size and style of selected text.
b. The default color of text is balck.
c. The setup of page such as margin size, orientation can be changed but can not be view on screen.
d. We cannot insert any picture in the document.
e. The text cannot be arrange.

Steps to start Notpad.

1. Go to window and write notepad than press enter.
2. Start -> all programme -> accessories -> notepad

Menu bar of Notepad

There are 5 menu in the menu bar of notepad. Each menu has different options in it.

1. File menu:-

There are seven options in it.

a. New [ ctrl + n ]:-
It is used to create a new document.

Note:
a. When we press ( ctrl + n )than a option is asked for previous unsaved document to save for don’t save.
b. In case of window 10, we can open the multiple document by pressing [ ctrl + shift + n ].
c. Open ( ctrl + 0 ):- This tool is used to open the saved document of notepad.

Note:- To open any document , we must remember the location , folder and file name.

2. Save [ ctrl + s ] :
To saved the document for future purpose.

Steps to save the document:-
1. Press [ ctrl + s ] or select save from file menu.
2. Select directory name.
3. Open / create folder.
4. Write file name and press Enter.

2. Edit Menu

There are 11 options in this menu.

1. Undo [ ctrl + z ] :
it is used to go one step backword and forwared,

2. Cut [ ctrl + x ] :
To cut the selected text of document.

3. Copy [ ctrl + c ]:
To copy the selected text of document.

4. Paste [ ctrl + v ]:
To paste the copied or cut text of document.

Note:
1. To copy or cut any text, we need to select the text.
2. Once copied/ cut text can be paste many times.

5. Find [ ctrl + f ]:-
To find any particular word from the document.

Note:
We need to choose direction to find any word in notepad.

6. Find Next [ F3 ]:-
To find the same next word of the document.

7. Replace [ ctrl + h ]:-
To exchange any old word with new word.

Note:
We write old word in upper box and new word in lower box and then click on replace it.

8. Go to [ ctrl + g ]:
using this option we can directly go to the any particular line by writing the line number.

Note: If go to does not work then click on “word wrap from format menu.

9. Select all [ ctrl + a ]:-
To select all the text of document.

10. Date and time [ F5 ]:
to write current date and time in the document.

11. Delete: -
To delete the selected text of document.

Note: In windows 10, two new options are also given.
a. Search [ ctrl + E ]:
To know the details of selected text using different options.
b. Find previous:-
To find the previous word of the document press .

3. Fromat Menu

There are two options in it.

1. Word wrap:-
If we select this option line in automatic change without pressing enter key.

2. Font:-
This option is used to perform the text related work such a.
a. Change text size
b. Change text style
c. Apply bold, italic effect.

Note:-
1. There is no size limitation for the text.
2. There are different font style to write in English.
Ex: - Calibri , times new roman , Algerian, monotype, Jokerman, elephant, Kokila, forte, ......... 3. We used “Krutidev” font style to write in Hindi.
4. We used wingding font style to write in symbolic language.

4. View Menu

This menu is generally used to show/hide status bar from screen.
Note:
1. Status bar display the line number.
2. In windows 10 , “Zoom “ options is also given.

5. Help Menu

There are two options in it.

a. About notepad:-
To display the version of notepad.

b. View help:-
To provide solution of different questions ,related to notepad.

Note:
windows 7 provide help and windows 10 provide online help.
