Microsoft Excel 2013

It is a spread sheet software in which we can create any types of data in the form of Row and Column. When we open excel the working area in form of cell.

At present excel is used to perform the various task such as:-
a. To create different types of list.
b. To create marksheet report.
c. To create sales/purchase record in shop/ company
d. To perform basic mathematical calculation.
e. To perform offline data entry work.

SOME IMPORTANT CONCEPT OF EXCEL:-

1. The document of excel is called workbook and the working area is called worksheet.

2. By default there is one sheet in the book that can be extended by clicking on ‘t’ symbol from status bar.

3. The width of column and height of Row can be changed be dragging the mouse from its heading after appearing of <-|-> | symbol.

4. Row number is indicated by 1,2,3,………….. and column number is indicated by A, B, C………Z, AA, BB, AC,…….. ZZ, AAA, AAB…. XFD.

5. There are ( 1048576 ) rows and ( 16384 ) column in a sheet of Excel 2013.
Note: - In excel 2013. There are 655536 rows and 256 columns.

6. We can directly go to any cell by writing cell name ( column * row ) in the name box ( below clipboard ).

7. The selected cell is called active cell that is indicated by green outline.
Note: The data that we enter is always written in active cell.

8. Continuous selected cell is called cell reference that is indicated as B2:D5.

9. To perform any types of text related work in excel we select cell not the text.

10. The sheet of excel can be view in three form.
a. Normal view ( As a sheet )
b. Page layout view ( As a page )
c. Page break view ( According to data )

11. To write the serial number , we just write 1,2 in respective cell and than drag the mouse to bottom by selecting the two data.

12. To write same data in the other cell of same column we just write first letter of the data.

13. To write the same data in same column we drag the mouse to the bottom.

14. To edit the data of cell, Double click on the cell or press [ F2 ] key.

15. To change the line in cell, press [ Alt + enter ]

16. We must use middle center alignment and border in the cell before printing the data of excel sheet.

17. To adjust the large data in a cell without increasing column width, we click on wrap text option from Home Menu.

18. Formula bar ( fx ) displays the complete data on the cell and it also display the formula used in the cell of sheet.

19. The copied cell appear as green moving dotted line and we can paste the copied cell data until dotted line blink.

HOME MENU

There are seven sub-menu that helps to perform all the data entry related work in excel sheet.

1. Clip-board sub-menu:

There are 4 options in it.
a. Cut [ ctrl + x ]
b. Copy [ ctrl + c ]
c. Paste [ ctrl + v ]
d. Format pointer [ No shortcut work like word ]

2. Font sub-menu :

There are only 10 tools in this sub-menu.
a. Font style
b. Font size
c. Shrink font size
d. Grow font size
e. Border
f. Bold
g. Italic
h. Underline
i. Font color
j. Paragraph

Note:-

1. The maximum font size is 409 and minimum is 1.

2. There is only 2 types of underline in Excel single underline and double underline.

3. The color of underline is as same as a text color. Its color cannot be change separately.

4. Before applying any border, we should select line style and line color.

5. To apply Superscript, Subscript and strikethrough we press [ ctrl + shift + f ] and select the required option.

3. Alignment Submenu

There are 11 tools in this submenu.
a. Alignment

There are 6 types of alignment that align the text at 9 different position of cell.
a. Top
b. Middle
c. Bottom
d. Left
e. Center
f. Right

b. Indent:

There are 2 types of Indents.
1. Left Indent.
2. Right Indent.

c. Merge
