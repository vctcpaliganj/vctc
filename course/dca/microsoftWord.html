Microsoft Word 2013

It is the best word processing software in which different types of documentation works are done.

We can perform various tasks in word such as : -
1. To write latter or application.
2. To create question paper and assignment.
3. To create a simple poster.
4. To create data list in the form of table.
5. To create and design graph, smart art, and attractive text.

Special features of word 2013.

1. Some templates are pre-designed in 2013.
2. Some additional menu is also provided to design the inserted object ( table, shape, picture, graph ).
3. We can easily convert the document of word in .pdf without using any software.
4. Spelling check, antonyms, synonyms and translate features are provided.
5. We can protect the document with password.

• Press F12 key and then click on “save as type “ option Now select pdf and press Enter.
• Click on file menu and then select “Export /publish”option . Click on “create a PDF “ then press enter.
• Press ctrl + p and then click on printer name. Now select “print to PDF “option. Click on print then write name and press enter.
• Print any document we need to select the printer name and write number of copy.
• To print alternate pages, we manually write the page no. separated by commands of dash ( - ).
• To print the both side of paper, we should use odd page or even page option.
• We can also print multiple page in a single sheet using “page per sheet “option.

File Menu

There are different options in this menu.

a. Info

It display the complete details of document such as size, date-time, total number of page and word, document name, etc.

b. New [ ctrl + n ]

To create a new document without closing the previous

Note:
We can open multiple document in word 2013.

c. Open [ ctrl + o ]

To open the existing document of word.

Note:
1. All the recently used document is displayed to the right side of open Windows.
2. To open any document we need to click on Computer then click on browse and finally select the dirve.

d. Save [ ctrl + s ]

To save the document for future purpose.

Note:
During saving the document we need to click on browse options to select the directory.

e. Save as [ F12 ]

To saved the prestored documents with other name after editing the previous document.

f. Print [ ctrl + p ]

To print any document on paper using printer.
1. We need to select the active printer name and write number of copy.
2. We can print alternate page by writing the page number separated by comma (,) and dashed ( - )
Example:
a. 1,3,7, 19
b. 5-10 5 to 10 page
3. To print both side of picture we should select odd page or even page option by clicking on “print all page”.
4. To print multiple page in a single sheet click on “page per sheet option”.
5. During printing the document we can also change the setup of page such as margin, size and orientation.

g. Publish/Export:-

Using this option we can convert the document of word into .pdf
PDF = Portable Document Format

Note:
The document of word can also be save in pdf format using save as type option.

Clear Formatting: // image
This tool remove all the applied style of the text and change the text in its original form.

Change Case : [ shift + F3 ] // image
There are five types of change case that convert small letter into capital and capital into small.
a. Sentence Case : First letter capital and rest is small.
b. Upper Case: All letter Capital
c. Lower Case: All letter Small
d. Capitalize each word: First letter of each word is capital.
e. Toggle Case: First letter of each word is small and rest is capital.

Text Effect : // image
There are different types of style for the text. We can select any one style.
The text can also be design using following options of this tool.
a. Outline
b. Glow
c. Shadow
d. Reflection
• We can also apply double underline in the text by pressing ctrl + shift + d
• There are different types of underline such as: single, double,dotted,wavy,thick etc.
• By default the color of underline is same as text color but we can seperately change color of underline.
• We can fill mixture of color ( gradient ) in the selected text.
• To increase or decrease one size of font we press [ ctrl + [ , ] ]

3. Paragraph Submenu:

There are 13 tools in this submenu to arrange in text properly: // image

a. Alignment

There are 4 types of Alignment
1. Left Alignment [ ctrl + L ]
2. Right Alignment [ ctrl + R ]
3. Centre Alignment [ ctrl + E ]
4. Justify Alignment [ ctrl + j ]

b. Indent

There are two type of indent that move the text to the left or right step by step.
1. Left indent
2. Right indent

c. Line spacing

To adjust the space between two lines.

d. Bullet

Bullet is a symbolic point that is added to the starting of line.
There are different types of Bullets that can be use in word after/before writing the text.
Ex: image of bullet.

Note:
we can also create a new Bullet by following steps/
1. Click on arrow /// image of Bullet tool
2. Select “Define New Bullet” option.
3. Click on “symbol” & then select “wingding” style.
4. Select any one Bullet & press Enter.

e. Numbering

This tool is used to automatic apply the serial number in each line of document.

Note:
1. We can also custom set the value of numbering by clicking on “set numbering value” option.
2. The space between numbering and text can be adjust by dragging the mouse from ruler.
3. The applied bullet and numbering can be move to the left of right by dragging the mouse.
4. As similar to text, we can also change the size, style, color and effect of the Bullet and Numbering.

f. Sorting // image

a. It is used to arrange the selected data in ascending of descending order.

g. Shading

To fill color in the line where cursor blink.

h. Border

There are different types of border such as : left, right , top , bottom, outside, all etc.
Border can be apply in the line using line style and line color.

i. Paragraph Indicator

It is used to show the end of paragraph.

4.. Style Submenu

There are different types of pre-formed style for the text
We can select any one style for the selected text.

We can also create a new style in style gallery by following steps:-

a. Click on dropdown arrow //image of style sub menu.
b. Click on ‘create a style’ option.
c. Write style name & then click on modify.
d. Set font size, style, color, Alignment & click on OK
