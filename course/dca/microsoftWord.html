Microsoft Word 2013

It is the best word processing software in which different types of documentation works are done.

We can perform various tasks in word such as : -
1. To write latter or application.
2. To create question paper and assignment.
3. To create a simple poster.
4. To create data list in the form of table.
5. To create and design graph, smart art, and attractive text.

Special features of word 2013.

1. Some templates are pre-designed in 2013.
2. Some additional menu is also provided to design the inserted object ( table, shape, picture, graph ).
3. We can easily convert the document of word in .pdf without using any software.
4. Spelling check, antonyms, synonyms and translate features are provided.
5. We can protect the document with password.

• Press F12 key and then click on “save as type “ option Now select pdf and press Enter.
• Click on file menu and then select “Export /publish”option . Click on “create a PDF “ then press enter.
• Press ctrl + p and then click on printer name. Now select “print to PDF “option. Click on print then write name and press enter.
• Print any document we need to select the printer name and write number of copy.
• To print alternate pages, we manually write the page no. separated by commands of dash ( - ).
• To print the both side of paper, we should use odd page or even page option.
• We can also print multiple page in a single sheet using “page per sheet “option.
